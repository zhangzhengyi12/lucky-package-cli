## 说明

npm 组件库 脚手架工具

## 安装

> npm i lucky-package-cli -g

## 使用

> lucky create <app-name>

## 开发

> npm run bootstarp # 安装所有依赖

> npm run dev:<template-name> # 测试某个模板

> npm run lucky # 运行主程序

模板的 package.json 生产与 dev 环境分离 模板的 package.json 可以随意更改
